cartBrowse

URL

/users/{id}/cart

GET

Response Type

Parameters

  • Path: id Integer

cartDelete

Changes the product or amount of a specific item.

URL

/users/{id}/cart/{item}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id Integer

    The id of the user

  • Path: item Integer

    The index of the item in the cart to modify

cartEdit

Changes the id or count of a specific item in the user's cart.

URL

/users/{id}/cart/{item}

HTTP Method

PATCH

Response Type

Parameters

  • Path: id Integer

    The id of the user

  • Path: item Integer

    The index of the item in the cart to modify
